# Meetdrip

Gestionale completo per magazzino/rivendita di prodotti cannabis.

## Stack Tecnologico

- **Backend**: Express.js + TypeScript
- **Frontend**: React + TypeScript
- **Database**: Neon (PostgreSQL serverless)
- **Hosting**: Render
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: Zustand
- **Data Fetching**: React Query

## Setup

### Prerequisiti

- Node.js (versione 18+)
- npm o yarn

### Installazione

1. Clona il repository
2. Copia `.env.example` in `.env` e configura le variabili d'ambiente
3. Installa le dipendenze:

```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

### Avvio

```bash
# Backend (porta 5000)
cd backend
npm run dev

# Frontend (porta 3000)
cd frontend
npm run dev
```
